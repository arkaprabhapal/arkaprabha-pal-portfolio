RFID Attendance System using Nodemcu 1.0

Details

RFID Attendance System

An RFID attendance system using ESP8266 is a project that uses radio frequency identification (RFID) technology to record the presence of projectsers or students.
You can check the video of the project here : Click here

  • The system consists of an ESP8266 development board, an MFRC522 RFID reader, and a web application (I used Google Sheets) that stores and displays the attendance data.

  • The ESP8266 connects to the internet via Wi-Fi and sends the scanned RFID card data to Google Sheets using HTTP requests.

  • The web application can also provide features such as time-in, time-out, and multiple gate entry options.

Details

Pantilt Security Camera

ESP32-Cam is used to capture photos/videos and control Pan-Tilt with the help of Web server.

  • Soft Access Point is used here. So the ESP32-Cam module creates its own wifi network.

  • Users can connect to the ip address by using any browser after connecting to the Wi-Fi network created by ESP32-Cam.

  • From the web server, the user can see the captured photos/videos and control the movement of the Pan tilt.
